Description

GitHub Action for adding, committing, and pushing changes to a repository.

Inputs

| name | description | required | default | | --- | --- | --- | --- | | github_token |

GitHub token

| true | "" | | message |

Commit message

| false | Save changes | | git_user_name |

Git user name

| false | pakkunbot | | git_user_email |

Git user email

| false | pakkunbot@users.noreply.github.com |

Runs

This action is a composite action.

Usage

```yaml - uses: PROJECT@VERSION with: github_token: # GitHub token # # Required: true # Default: ""

message:
# Commit message
#
# Required: false
# Default: Save changes

git_user_name:
# Git user name
#
# Required: false
# Default: pakkunbot

git_user_email:
# Git user email
#
# Required: false
# Default: pakkunbot@users.noreply.github.com

```
